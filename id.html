Micro API

id

Expected types: String Number

Description:

A unique value used for identifying a resource. For identifying an individual resource, it should be a singular value, but for identifying a reference, it MAY be an array. For example:

{
  "@context": {
    "@vocab": "/#",
    "µ": "http://micro-api.org/"
  },
  "@graph": [ {
    "@type": "Movie",
    "@id": "/movies/the-matrix",
    "µ:id": "the-matrix",
    "name": "The Matrix",
    "actor": {
      "@id": "/movies/the-matrix/actors",
      "µ:id": [ "keanu-reeves", "laurence-fishburne" ]
    }
  } ]
}
