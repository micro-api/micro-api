Micro API

query

Expected types: Object

Description:

A container for showing information about the current query. It MUST exist only at the top-level document. For example, it may advertise query options available in the query string.

{
  "query": {
    "@context": null,
    "include": [],
    "sort": {},
    "field": {},
    "match": {},
    "limit": 1000,
    "offset": 0
  }
}
